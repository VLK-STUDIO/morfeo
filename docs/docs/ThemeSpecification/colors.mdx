## Interface

### Structure

```typescript
interface Colors {
  [key: string]: string;
}

type Color = keyof Colors;
```

### Usage

```typescript
// Types are re-exported for ease of use by other packages
// like @morfeo/core, @morfeo/web or @morfeo/react
import { Color, Colors } from '@morfeo/spec';
```

## Color Placeholders

Color placeholders allow referring to another theme color by prefixing it with the character `$`:

### Example 
```typescript
const colors: Colors = {
  // main palette
  dark: '#000000',
  light: '#ffffff',
  primary: '#06f',
  secondary: '#db2494'
  // functional palette
  background: '$light', // it will be resolved as #ffffff
  foreground: '$dark', // it will be resolved as #000000
  borderColor: '$primary', // it will be resolved as #06f
  ctaColor: '$secondary' // it will be resolved as #db2494
}
```

## Extend Slice with custom properties

If you're using typescript and you add custom colors to your theme configuration, you'll need to merge
the default `Colors` interface with your custom one.
We suggest creating a declaration file (for example: `morfeo.d.ts` or `types.d.ts`) like the following one:

```typescript
import { colors } from './path/to/your/custom/colors';

type MyColors = typeof colors;

declare module '@morfeo/spec' {
  export interface Colors extends MyColors {}
}
```

Read more about how to extend the default Morfeo Theme [here](../Features/extendible.mdx).
